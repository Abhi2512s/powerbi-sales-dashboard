# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview

This project presents a Sales Performance Dashboard built using Power BI.
It analyzes sales trends, profit, and growth using DAX and time intelligence.

---

## 🎯 Objectives

* Analyze monthly sales trends
* Compare performance with previous year
* Identify top-performing categories and regions
* Provide actionable business insights

---

## 📊 Key Features

* KPI Cards (Total Sales, Profit, Profit Margin, YoY Growth)
* Time Intelligence (YTD, Previous Year, YoY Growth)
* Interactive Filters (Year, Category, Region)
* Category & Regional Analysis

---

## 📈 Insights

* Sales peak in Q4 indicating strong seasonal demand
* West region contributes highest revenue
* Technology category drives maximum sales
* Furniture shows lower profit margin

---

## 🛠 Tools Used

* Power BI
* DAX
* Data Modeling

---

## 📂 Files

* `.pbix` file → Power BI Dashboard
* Dataset used for analysis

---

## 🚀 How to Use

1. Download `.pbix` file
2. Open in Power BI Desktop
3. Interact with filters and visuals
